# meta-rule-creator

현재 프로젝트에 맞는 지침 유지보수(메타) 문서를 `rules/meta.md`에 생성·갱신합니다.

## 생성 절차

1. **프로젝트 스캔** — 기존 지침 구조를 파악합니다.
   - `rules/` 하위 파일 목록.
   - `CLAUDE.md`에서 참조하는 지침 문서.
2. **내용 구성**
   - 같은 디렉토리의 `meta.md` 전체를 기본 내용으로 사용합니다.
   - 프로젝트 고유의 지침 배치 규칙이나 리뷰 주기가 있다면 "## 이 프로젝트의 세부 지침"에 추가합니다.
3. **파일 기록**
   - 대상: `rules/meta.md`
   - 상위 디렉토리 부재 시 함께 생성.
   - 이미 존재하면 덮어쓰지 않고 diff 후 사용자에게 확인.

# context-rule-creator

현재 프로젝트에 맞는 컨텍스트 관리 지침 문서를 `rules/context.md`에 생성·갱신합니다.

## 생성 절차

1. **프로젝트 스캔** — 컨텍스트 맥락을 파악합니다.
   - 레포 크기와 대용량 파일 위치.
   - `rules/` 구성, `CLAUDE.md` 크기.
   - 에이전트·메모리 시스템 사용 여부.
2. **내용 구성**
   - 같은 디렉토리의 `context.md` 전체를 기본 내용으로 사용합니다.
   - 프로젝트 고유의 대용량 파일 취급 기준이나 위임 정책을 "## 이 프로젝트의 세부 지침"에 추가합니다.
3. **파일 기록**
   - 대상: `rules/context.md`
   - 상위 디렉토리 부재 시 함께 생성.
   - 이미 존재하면 덮어쓰지 않고 diff 후 사용자에게 확인.
